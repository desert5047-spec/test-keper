# 配布前チェック（MVP）

## 基本動作
- 初回起動 → ホームが表示される
- ログインができる（メール）
- ログアウトができる

## 記録フロー
- 写真を撮影/選択できる
- 記録を保存できる
- 保存後の確認画面が表示される
- 「続けて記録する」でフォームがリセットされる
- 「ホームに戻る」でホームに遷移できる

## 表示系
- 一覧が表示される
- 月別集計が表示される
- 教科/点数/スタンプが表示される
- 画像がある記録は写真が表示される

## 権限・クラッシュ
- iOSのカメラ/写真権限ダイアログが出る
- 画像拡大/縮小が動作する
- クラッシュが発生しない
